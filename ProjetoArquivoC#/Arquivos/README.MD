# README

Sobre o manipulador de arquivos TXT;
Este projeto tem como objetivo cadastrar pessoas com CPF ou com CPNJ ao lado de seus nomes e 
salvá-los em um arquivo Txt.

*Funcionalidades:

*Receber informações do usuário pelo terminal.
*Alterar dinamicamente os tipos de valores exigidos por meio de variáveis recebidas pelo usuário no terminal.
*Mostrar resumo das informações salvas no termina.
*Salvar no arquivo txt.

Tecnolog8ias utilizadas:

*Microsoft Visual Studio
*ASP.NET
*SDK DO WINDOWS 11
*Microsoft Visual Code

Organização do projeto:

*Arquivos do tipo CS permanecem na mesma pasta para projetos pequenos, facilitar o caminho da referência.

Pré-requisitos de instalação:

*Como não é um programa empacotado, não há instalação, apenas execução e depuração.

Execução da aplicação:

*Para executar o projeto basta abrir o terminal na pasta do executável e digitar: dotnet run "nomeDoArquivo.CS".

*Atualmente não há erros de execução.
